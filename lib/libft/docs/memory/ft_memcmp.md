# ft_memcmp

## Description

Compare deux zones mémoire

## Prototype

```c
int ft_memcmp(const void *s1, const void *s2, size_t n);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s1` | `const void *` | Première zone |
| `s2` | `const void *` | Deuxième zone |
| `n` | `size_t` | Nombre d'octets |


## Valeur de retour

0 si égales, <0 si s1 < s2, >0 si s1 > s2

La fonction `ft_memcmp` compare deux zones mémoire
