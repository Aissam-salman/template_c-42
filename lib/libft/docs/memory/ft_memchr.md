# ft_memchr

## Description

Localise un octet dans une zone mémoire

## Prototype

```c
void *ft_memchr(const void *s, int c, size_t n);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s` | `const void *` | Zone mémoire |
| `c` | `int` | Octet à trouver |
| `n` | `size_t` | Nombre d'octets à examiner |


## Valeur de retour

Un pointeur vers l'octet trouvé ou NULL
