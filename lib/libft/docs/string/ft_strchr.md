# ft_strchr

## Description

Localise la première occurrence d'un caractère dans une chaîne

## Prototype

```c
char *ft_strchr(const char *s, int c);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s` | `const char *` | Chaîne à parcourir |
| `c` | `int` | Caractère à trouver |


## Valeur de retour

Un pointeur vers le caractère trouvé ou NULL
