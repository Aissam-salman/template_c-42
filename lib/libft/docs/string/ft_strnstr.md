# ft_strnstr

## Description

Localise une sous-chaîne dans une chaîne limitée

## Prototype

```c
char *ft_strnstr(const char *s1, const char *s2, size_t n);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s1` | `const char *` | Chaîne où chercher |
| `s2` | `const char *` | Sous-chaîne à trouver |
| `n` | `size_t` | Nombre maximum de caractères à examiner |


## Valeur de retour

Un pointeur vers le début de la sous-chaîne trouvée ou NULL

## Description détaillée

La fonction `ft_strnstr` localise une sous-chaîne dans une chaîne limitée
