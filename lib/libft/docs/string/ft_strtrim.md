# ft_strtrim

## Description

Supprime les caractères spécifiés au début et à la fin d'une chaîne

## Prototype

```c
char *ft_strtrim(char const *s1, char const *set);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s1` | `char const *` | Chaîne à trimmer |
| `set` | `char const *` | Ensemble de caractères à supprimer |


## Valeur de retour

La chaîne trimmée allouée ou NULL en cas d'erreur
