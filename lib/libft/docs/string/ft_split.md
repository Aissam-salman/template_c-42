# ft_split

## Description

Divise une chaîne en tableau de chaînes selon un délimiteur

## Prototype

```c
char **ft_split(char const *s, char c);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s` | `char const *` | Chaîne à diviser |
| `c` | `char` | Caractère délimiteur |


## Valeur de retour

Tableau de chaînes alloué ou NULL en cas d'erreur
