## Description

Extrait une sous-chaîne d'une chaîne

## Prototype

```c
char *ft_substr(char const *s, unsigned int start, size_t len);
```

## Paramètres

| Paramètre | Type | Description |
|-----------|------|-------------|
| `s` | `char const *` | Chaîne source |
| `start` | `unsigned int` | Index de début |
| `len` | `size_t` | Longueur maximale de la sous-chaîne |


## Valeur de retour
